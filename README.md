# Pedra, Papel e Tesoura ✊📄✂️

Este projeto é uma atividade acadêmica em **C++** que simula o clássico jogo **Pedra, Papel e Tesoura**.  
O programa recebe as escolhas de dois jogadores e aplica as regras tradicionais para determinar o vencedor ou se houve empate.

## 🎯 Objetivo
Permitir que dois usuários joguem Pedra, Papel e Tesoura diretamente no terminal, com base nas seguintes regras:

- **Pedra (1)** vence **Tesoura (3)**
- **Tesoura (3)** vence **Papel (2)**
- **Papel (2)** vence **Pedra (1)**
- Valores iguais resultam em **Empate**

## 🚀 Como executar
1. Copie o codigo e execute como arquivo .cpp
2. ou copie e cole em um arquivo que compile em C++

## 📌 Exemplo de uso
=====Pedra, Papel e Tesoura=====
Turno do: Jogador 1
Escolha:
[1]Pedra
[2]Papel
[3]Tesoura
1

=====Pedra, Papel e Tesoura=====
Turno do: Jogador 2
Escolha:
[1]Pedra
[2]Papel
[3]Tesoura
3

=====Pedra, Papel e Tesoura=====
O jogador 1 venceu!
Pedra ganha de Tesoura.
